# Privacy Policy & User Instructions (GDPR Compliant)

Цей документ визначає правила використання проєкту, обмеження відповідальності та політику конфіденційності згідно з вимогами GDPR.

## 1. Загальні положення (EULA)
* Цей проєкт є навчальною роботою з предмета "Конструювання програмного забезпечення" (КОП).
* Програмне забезпечення надається за принципом "як є" (as is) без будь-яких гарантій.
* Автор не несе відповідальності за будь-які збитки, що виникли внаслідок використання цього ПЗ.

## 2. Інструкція для користувача
* Проєкт призначений для демонстрації навичок розробки (Storybook, JSDoc, Cookie Popup).
* Для запуску проєкту локально використовуйте команди, описані у файлі `README.md`.
* Для перегляду компонентів запустіть Storybook (`npm run storybook`).

## 3. Політика конфіденційності (GDPR)
Ми серйозно ставимося до захисту ваших даних. Цей розділ пояснює, як ми обробляємо інформацію:

### 3.1. Які дані ми збираємо?
* Цей проєкт **не збирає** персональні дані (імена, електронні адреси, номери телефонів).
* Ми використовуємо лише технічні файли **Cookie** для збереження вашого вибору щодо згоди на обробку даних.

### 3.2. Використання файлів Cookie
* При вході на сайт з'являється **Cookie Popup**.
* Якщо ви натискаєте **"Прийняти"**, у вашому браузері зберігається мітка про згоду.
* Якщо ви натискаєте **"Відхилити"**, жодні сторонні скрипти не завантажуються, а ваш вибір ігнорується.

### 3.3. Ваші права згідно з GDPR
* **Право на доступ:** Ви можете переглянути цей документ у будь-який момент.
* **Право на забуття:** Ви можете очистити файли cookie у налаштуваннях свого браузера в будь-який час, щоб видалити дані про свій візит.

## 4. Обмеження
* Заборонено використовувати цей код у комерційних цілях без посилання на автора.
* Проєкт не призначений для обробки реальних фінансових або конфіденційних даних.
